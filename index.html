Exam questions should be submitted as a plain text file (.txt) using the following structure:

[Q001] What is the maximum number of cameras supported per Archiver?
TRUE The maximum is 200 cameras per Archiver
FALSE 500 cameras per Archiver
FALSE 1000 cameras per Archiver
FALSE There is no limit

[Q002] Which protocol is used for secure communication between Security Center components?
FALSE HTTP
TRUE TLS
FALSE FTP
FALSE SMTP

Each question starts with a Question ID in brackets followed by the question text. Each answer is on its own line, prefixed with TRUE or FALSE.
